## How to use it

### Creates a development environment and install the required dependencies

``` 
make install
```

### Updates the development environment
``` 
make update-env
```

### Starts a lightweight development Web server on the local machine
``` 
make serve
```

### Builds the documentation
``` 
make build
```

### Quicktart demo app
``` 
make quickstart
```
